(inner, constant) Roles :number

Enumeration for authority.
Type:
  • number
Properties:
Name Type Description
Developer number Only accessible for debugging and testing. Commands considered Work In Progress. Same user applications as Violator.
WorkspaceOwner number Is a Workspace Owner. Provide maximum access to bot features. Control all permissions, even admins'.
WorkspaceAdmin number Is a Workspace Admin. Provide access to everything but owner-specific features. Control all permissions.
Member number Is a standard member. Restrict to standard commands. Control permissions for SingleChannelGuests.
MultiChannelGuest number Guest. Restrict to very simple commands.
SingleChannelGuest number Guest. Potentially untrustable. Same as MultiChannelGuest, permissions can be managed by Member.
Violator number Violator. Do not accept ANY command requests. (Only given to users by a higher permission managing entity)
